Langkah langkah git

  • git init = untuk menginialisasi git di project kamu
  • git add . = untuk menambahkan file anda
  • git commit -m "pesan commit kamu" = tambahkan commit kamu
  • git remote add origin "remote repository url" = tambahkan url repository kamu
  • git remote -v = untuk melihat apakah remote sudah ditambahkan
  • git push origin main = gunakan untuk push git anda ke repository github
